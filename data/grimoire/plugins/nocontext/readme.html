nocontext

This is a relatively complex plugin.

It transforms any chosen playlist into desktop professional writing and typesetting app powered by ConTeXt.

Requirements

  • installed ConTeXt, added to system pathor the plugin could be modded to include full path to something_similar_to\tex\texmf-win64\bin\context.exe
  • text editor that will be used to edit .tex files

Customizations

Upon enabling this plugin, a custom entry will be added to the current playlist settings. It can be edited manually at any time. This way different playlists will be able to utilise this plugin differently.
